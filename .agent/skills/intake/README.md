# Intake Skill

Thu thập requirements từ user một cách có cấu trúc.

## Quick Start

```bash
# Bắt đầu intake session
node ".agent/skills/intake/scripts/start-intake.js"
```

## Output

File `output/intake/intake.md` chứa:
- Project overview
- Target users
- Features (P0, P1, P2)
- Constraints
- Open questions

## Integration

Sau intake, chuyển sang:
1. **Research Skill** - Tìm repo mẫu
2. **Spec-Kit Skill** - Tạo specifications

## Xem chi tiết

Đọc [SKILL.md](./SKILL.md) để hiểu workflow đầy đủ.
