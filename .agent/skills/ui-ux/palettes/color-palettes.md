## Dark Mode Variants

When converting to dark mode:
- Background: Invert to dark (#0F172A)
- Surface: Slightly lighter dark (#1E293B)
- Text: Light colors (#F1F5F9)
- Primary: Keep or slightly lighten
- Reduce contrast on borders
