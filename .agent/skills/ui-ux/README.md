# UI-UX Skill

Thiết kế UI/UX và tạo design system.

## Quick Start

```bash
# Generate design system
node ".agent/skills/ui-ux/scripts/generate-design.js"

# View design system
node ".agent/skills/ui-ux/scripts/view-design.js"
```

## Output

- `output/design/MASTER.md` - Global design tokens
- `output/design/pages/*.md` - Page-specific specs
- `output/design/handoff.md` - Developer handoff

## Design System Includes

1. **Colors** - Palette with semantic colors
2. **Typography** - Font family, type scale
3. **Spacing** - Consistent spacing system
4. **Components** - Buttons, cards, inputs
5. **Patterns** - Layout, grid, responsive
