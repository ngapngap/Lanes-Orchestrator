# Industry Reasoning Rules

## SaaS / Dashboard Applications

### Color Strategy
- Primary: Blue (#3B82F6) - Trust, professionalism
- Background: Light grays, white
- Accent: Purple/Indigo for premium feel

### Layout Rules
1. Data-first: Charts and metrics prominent
2. Navigation: Left sidebar (collapsible)
3. Actions: Top-right of each section
4. Whitespace: Generous padding

### Typography
- Font: Inter, SF Pro, system-ui
- Weights: 400 (body), 500 (labels), 600 (headings)
- Size: 14-16px base

### Component Patterns
- Cards for data grouping
- Tables with sorting/filtering
- Skeleton loaders for data
- Toast notifications

### Anti-patterns
- Avoid: Bright colors everywhere
- Avoid: Nested modals
- Avoid: Dense layouts without breathing room

## E-commerce

### Color Strategy
- Primary: Orange/Coral (#F97316) - Energy, urgency
- CTA: High contrast, action-oriented
- Trust: Green for price, blue for trust badges

### Layout Rules
1. Product focus: Large images, clear pricing
2. Navigation: Mega menus for categories
3. Cart: Always visible (sticky header)
4. Mobile-first: Touch-friendly

### Typography
- Font: DM Sans, Poppins
- Emphasis on prices and CTAs
- Clear hierarchy for product info

### Component Patterns
- Product cards with hover effects
- Quick view modals
- Infinite scroll for listings
- Breadcrumbs for navigation

### Anti-patterns
- Avoid: Cluttered product pages
- Avoid: Hidden prices
- Avoid: Slow image loading

## Healthcare

### Color Strategy
- Primary: Green (#10B981) - Health, calm
- Secondary: Blue (#3B82F6) - Trust
- Background: Very light greens, white
- Alert: Red only for emergencies

### Layout Rules
1. Clarity: Large text, clear actions
2. Privacy: Clear data handling info
3. Accessibility: AAA contrast ratios
4. Forms: Step-by-step, clear progress

### Typography
- Font: Inter, system-ui
- Size: 16-18px minimum
- High contrast required

### Component Patterns
- Clear form validation
- Progress indicators
- Appointment calendars
- Secure input fields

### Anti-patterns
- Avoid: Aggressive colors
- Avoid: Complex navigation
- Avoid: Small text/buttons

---

## Finance / Banking

### Color Strategy
- Primary: Navy/Dark Blue (#1D4ED8) - Trust, stability
- Accent: Green for positive, red for negative
- Background: Professional whites/grays

### Layout Rules
1. Security first: Clear secure indicators
2. Data clarity: Clean tables, clear numbers
3. Actions: Confirmations for transactions
4. History: Easy access to past activity

### Typography
- Font: Inter, SF Pro
- Monospace for numbers
- Clear data formatting

### Component Patterns
- Account cards
- Transaction tables
- Secure input with masking
- Two-factor authentication flows

### Anti-patterns
- Avoid: Playful design
- Avoid: Unclear action consequences
- Avoid: Hidden fees/information

---

## Creative / Portfolio

### Color Strategy
- Primary: Purple (#8B5CF6) or custom brand color
- Background: Can be dark or light
- Accent: Bold, contrasting colors

### Layout Rules
1. Visual first: Large images/videos
2. Minimal text: Let work speak
3. Unique: Stand out from templates
4. Animation: Subtle motion adds polish

### Typography
- Font: Display fonts for headings
- Creative font pairings
- Large, impactful titles

### Component Patterns
- Image galleries
- Project case studies
- Testimonials
- Contact forms

### Anti-patterns
- Avoid: Generic stock photos
- Avoid: Slow loading (optimize media)
- Avoid: Over-animation
